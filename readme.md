# DevCamper API

> Backend API for DevCamper applicaiton, whihc is a bootcamp directory website

## Usage

Rename the "config/config.env.env" to "config/config.env" and update the value/settings to your own

## Install Dependecies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

#Run in prod mode
npm start
```
